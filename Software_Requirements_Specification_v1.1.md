# Software Requirements Specification

## 1. Introduction
This document outlines the software requirements specification (SRS) for the algorithm snack bar project. The currency for all transactions will be in euros (EUR).

## 2. Currency References
All financial transactions should be expressed in euros (EUR). Any references to DKK currency have been removed from this document.

## 3. Payment Information
Payments for services rendered should be in EUR. Previous mentions of EUR/DKK have been replaced with EUR, ensuring clarity in currency representation.

## 4. Conclusion
This SRS has been updated to reflect the changes in currency references to ensure that all stakeholders acknowledge that transactions will occur exclusively in EUR. The removal of DKK references aims to streamline processes and reduce confusion.
